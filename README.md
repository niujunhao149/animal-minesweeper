# 🦁 动物扫雷

一个可爱的动物主题扫雷游戏，支持自定义 emoji 和图片！

## 🎮 游戏特色

- 🦁 地雷用狮子代替
- 🐶 旗帜用狗狗代替
- 😺/😿/😎 猫咪表情状态
- 支持三种难度：简单(9×9)、中等(16×16)、困难(16×30)
- 失败后延迟 2 秒显示弹窗，可以查看战局
- 触雷位置用红色背景高亮标记

## 🎨 自定义功能

点击右上角 ⚙️ 按钮打开设置面板：

- **自定义 Emoji**：每个状态有 6 个预设 emoji 可选
- **图片上传**：可以上传自定义图片替换 emoji
- **设置保存**：自动保存到浏览器本地存储

## 🎯 游戏玩法

- **左键点击**：揭开格子
- **右键点击**：标记旗帜
- 找出所有安全格子即可获胜！

## 🚀 在线游玩

直接打开 `minesweeper.html` 文件即可在浏览器中游玩。

## 📝 技术栈

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
